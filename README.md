# React 3D Glassmorphism Auth Page

A modern, interactive Login and Sign Up interface featuring a stunning frosted glass (glassmorphism) design floating over a live, animated 3D background. Built with React, Three.js, and Tailwind CSS.

## ✨ Features

* **Live 3D Background:** Smoothly animated, floating geometric shapes using React Three Fiber.
* **Glassmorphism UI:** A highly polished, transparent frosted glass card with deep blur and subtle borders.
* **Dynamic Auth Toggle:** Seamlessly switch between "Login" and "Sign Up" states within the same component.
* **Password Visibility:** Interactive toggle to show/hide the password field.
* **Brand Integrations:** Official SVG logos for Google, GitHub, and Facebook using `react-icons`.
* **Fully Responsive:** Scales beautifully across desktop and mobile screens.

## 🛠️ Tech Stack

* **Framework:** React 18 + Vite
* **Styling:** Tailwind CSS (v3)
* **3D Rendering:** Three.js
* **React 3D Integration:** `@react-three/fiber` & `@react-three/drei`
* **Icons:** `lucide-react` (UI icons) & `react-icons` (Brand logos)
